# Vite SPA Boilerplate

A minimal boilerplate for building single-page applications with Vite and vanilla JavaScript.

## Features

- Fast HMR (Hot Module Replacement) with Vite
- Modern JavaScript (ES modules)
- Optimized build process
- Development server with instant feedback

## Getting Started

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm or yarn

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

Create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

```
.
├── index.html          # Entry HTML file
├── main.js            # Main JavaScript file
├── style.css          # Global styles
├── package.json       # Project dependencies and scripts
└── README.md          # This file
```

## Customization

This boilerplate uses vanilla JavaScript, but you can easily extend it:

- Add a framework (React, Vue, Svelte)
- Add TypeScript
- Add CSS preprocessors (Sass, Less)
- Add linting and formatting tools
